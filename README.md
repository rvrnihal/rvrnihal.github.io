## 🔧 Customization

### Update Content
Edit `index.html`:
- Project details
- Skills and expertise
- Experience entries
- Contact information

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary: #000000;      /* Main color */
    --accent: #ff6b35;       /* Highlight color */
    --light: #f5f5f5;        /* Background */
}
```

### Add New Project
Copy project card in HTML and update:
- Project name
- Description
- Technologies
- Links
